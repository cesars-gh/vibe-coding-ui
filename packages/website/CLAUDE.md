This is an Astro project. Key information for making changes:

- Pages go in `src/pages/` — each `.astro` file becomes a route (e.g., `about.astro` → `/about`)
- Layouts go in `src/layouts/` — `Layout.astro` is the base HTML shell
- Components go in `src/components/`
- Static assets go in `public/`
- Styles can be in `<style>` tags within `.astro` files (scoped by default)
- Use Astro's component syntax: frontmatter in `---` fences, HTML/JSX-like template below
- The dev server has HMR — changes appear instantly in the browser
